# ERP Farmacia ARMONÍA

Sistema de gestión completo para Farmacia ARMONÍA desarrollado fullstack con:
- **Backend**: NestJS + TypeScript + PostgreSQL (TypeORM)
- **Frontend**: React 18 + TypeScript + Vite
- **Base de datos**: Migraciones con Flyway
- **Autenticación**: JWT + Roles + Guards

## Módulos incluidos
- Ventas y facturación
- Control de inventario y lotes
- Kardex completo
- Alertas de stock bajo y vencimientos
- Gestión de clientes, proveedores, productos
- Auditoría de acciones
- Devoluciones
- Reportes y dashboard
